# Assets Directory

This directory contains project assets organized by type and purpose.

## Directory Structure

### `images/startup/`
Contains startup screen images in various formats and sizes:
- `new_startup_bitmap.png` - Original high-resolution startup image
- `startup_140x40.png` - Startup image resized to 140x40 pixels
- `startup_160x40.png` - Startup image resized to 160x40 pixels (current display size)

### `images/screenshots/`
Contains development screenshots and visual documentation:
- `screenshot_resized.png` - Resized screenshot for documentation
- `screenshot-De8lylrp.png` - Original screenshot from development

## Usage

The startup images are used by the Flash programming tools to create the startup splash screen displayed on the device. The 160x40 pixel version matches the current display hardware specifications.

Screenshots are kept for development reference and documentation purposes.
