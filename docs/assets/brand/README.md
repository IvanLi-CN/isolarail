# IsolaRail Brand Assets

This directory contains the project brand image assets.

The complete inventory, intended usage, dimensions, file sizes, and media previews are tracked in
[`docs/specs/b8r3n-brand-visual-assets/SPEC.md`](../../specs/b8r3n-brand-visual-assets/SPEC.md).

## Files

- `isolarail-logo-lockup.png`: primary logo lockup.
- `isolarail-logo-lockup.svg`: true SVG geometry for the primary logo lockup.
- `isolarail-logo-lockup-light.svg`: true SVG geometry for the light theme logo lockup.
- `isolarail-logo-lockup-dark.svg`: true SVG geometry for the dark theme logo lockup.
- `isolarail-logo-lockup-light.png`: transparent-background logo lockup for light theme.
- `isolarail-logo-lockup-dark.png`: transparent-background logo lockup for dark theme.
- `isolarail-app-icon.png`: square app icon with a transparent canvas and rounded tile.
- `isolarail-app-icon-light.svg`: SVG source for the light theme app icon export.
- `isolarail-app-icon-dark.svg`: SVG source for the dark theme app icon export.
- `isolarail-app-icon-light.png`: transparent-canvas app icon for light theme.
- `isolarail-app-icon-dark.png`: transparent-canvas app icon for dark theme.
- `isolarail-poster.png`: portrait poster image.
- `isolarail-github-social-preview.png`: GitHub social preview image.
- `isolarail-github-social-preview-variant.png`: alternate GitHub social preview image.
- `isolarail-product-scene-portrait.png`: portrait product scene source for future HTML poster layouts.
- `isolarail-product-scene-wide.png`: wide product scene source for future HTML poster layouts.

## Usage

Use the logo lockup as the primary identity mark on documentation and project pages.
Use the app icon where a square icon is required.
Use the light and dark theme assets for runtime favicon, touch icon, manifest icon, and themed UI exports.
Use the GitHub social preview image for repository social cards.
Use the GitHub social preview variant as an alternate social card or campaign preview.
Use the product scene images as source visuals for generated or HTML-composed poster layouts.

Keep the selected logo geometry stable unless the brand direction is explicitly reopened.
